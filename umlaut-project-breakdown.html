StatsTeam Size: 5
Project Length: 4 weeks
Engine: Unity
Constraints
Roughly 3 days to come up with a pitch for the game
I was the only person comfortable working with code and implementing the mechanics
No words are allowed to be used in the game (except for the credits portion) as the main theme of the project.

Process

Brainstorm

Due to the time constraints, most of the game did not deviate far from the original pitch I made within those 3 days.

Role Asssignment

Since everyone except for myself was interested in making the art for the game, roles were mostly divvied up between the artist as to which section they would like to work on, eg. characters, environment, UI etc.

Develop

Each person works on their assigned tasks individually and reach out to each other whenever help is needed.

Staying on Track

The game was pretty straight forward mechanics wise. But the art team faced some trouble keeping the style consistent across all discipline and a lot of effort was made to keeping everyone on the same page by establishing a heavily simplified art bible.
